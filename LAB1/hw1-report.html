Execution description

Quick Start

  • 透過 pip install -r requirements.txt 安裝相關環境
  • 執行 main.py，以執行作業所有小題

Part 1. Build Data

  • 用 build_data 來建立資料
  • 預設建立資料筆數為 1000 筆
  • xx 為利用 np.random.uniform 生成，數值範圍為 0 到 1000
  • 與 benchmark 的間隔 rr 為利用 np.random.uniform 生成，數值範圍為 1 到 1000

Part 2. PLA

執行 prob_2.py，以執行 part 2

  1. 用 PLA_3_times_with_30_data 跑所有程式
  2. 總共跑3次
    1. build_data來建立資料集
      • xx 為 (1,x,y)(1,x,y) 的組合
      • yy 為對應 label 之組合
    2. 預設 ww 為對應 (1,x,y)(1,x,y) 之參數，初始為亂數生成
    3. plt_proc 為畫圖的相關程式，故不多解釋
    4. PLA() 為執行 PLA algorithm 的程式
      1. 先產生一條亂序、無重複、從0到資料數量的整數序列
      2. 對該條序列依序檢測
      3. get_sign() 為取得 sign(wT∗x)sign(wT∗x)
      4. 如果 sign(wT∗x)≠yisign(wT∗x)≠yi
        • 更新 w=w+yi∗xiw=w+yi∗xi
    5. 利用 verification() 驗證結果
    6. 更新總迭代次數，並輸出圖片
  3. 取得平均迭代次數

Part 3. PLA v.s. Pocket

執行 prob_3.py，以執行 part 3

  1. 用 pocket_vs_pla 跑所有程式
  2. build_data來建立資料集
    • x 為 (1, x, y) 的組合
    • y 為對應 label 之組合
  3. 預設 w 為對應 (1, x, y) 之參數，初始為亂數生成
  4. plt_proc 為畫圖的相關程式，故不多解釋
  5. 利用 time.time() 取得當下秒數
  6. 利用 PLA() 執行 PLA
  7. 利用 verification() 驗證結果及計算錯誤率
  8. 利用 Pocket() 執行 Pocket
    1. iteration 為設定總共要跑幾回，預設為 10000
    2. continuous_threshold 為設定如果連續沒有更新 ww 幾次要停止繼續遞迴，預設為 1000
    3. 每回合都隨機產生一個 index
    4. 用 get_sign() 判斷是否正確
    5. 分別利用 verification() 取得舊參數與新參數的錯誤率
    6. 進行比較並更新參數
  9. 最後輸出結果

Part 4. Pocket with linear inseparable

執行 prob_4.py，以執行 part 4

  1. 用 pocket_with_inseparable 跑所有程式
  2. build_data來建立資料集
    • x 為 (1, x, y) 的組合
    • y 為對應 label 之組合
  3. 利用 np.random.choice 來隨機選擇要 mislabel 的 index 值
  4. 預設 w 為對應 (1, x, y) 之參數，初始為亂數生成
  5. plt_proc 為畫圖的相關程式，故不多解釋
  6. 利用 time.time() 取得當下秒數
  7. 利用 Pocket() 執行 Pocket
    1. iteration 為設定總共要跑幾回，預設為 10000
    2. continuous_threshold 為設定如果連續沒有更新 ww 幾次要停止繼續遞迴，預設為 1000
    3. 產生一條亂序、無重複、從0到資料數量的整數序列
    4. 對該條序列依序檢測
    5. 用 get_sign() 判斷是否正確
    6. 分別利用 verification() 取得舊參數與新參數的錯誤率
    7. 進行比較，如果更新後權重 wtwt 小於最佳權重 wbestwbest，則更新 wbestwbest
  8. 最後輸出結果
